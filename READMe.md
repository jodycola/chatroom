# Fullstack Real-Time Chatroom

### Features
<ul>
  <li> Create a user </li>
  <li> Join different chatrooms </li>
  <li> Chat with people in real-time </li>
</ul>

### Setup
First you must install the necessary files.  

##### Rails
`CD` into `chatroom_api`  
run `bundle install`
run `rails s`

##### React
`CD` into `client`
run `npm install`
run `npm start`

##### Node.js
`CD` into `server`
run `npm install`
run `node index.js`
